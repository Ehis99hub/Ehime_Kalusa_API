# Ehime_Kalusa_API

## Setup Instructions
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm test` to execute the tests.

## npm version
- 10.7.0
## node version
- v22.2.0
## cucumber js version
- 11.1.0

## Dependencies
- Node.js and npm
- @cucumber/cucumber
- axios
- chai
- ts-node
- javascript

## Run test
- npm test
## Run test using cucumber
- npx cucumber -js
